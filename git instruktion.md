## Списки

Чтобы добавить ненумерованные списки нужно выделить текст звёздочкой (*) в начале строки. Например, вот так:

* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки нужно просто пронумеровать. Например, вот так:

1. Первый пункт
2. Второй пункт
